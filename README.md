# 💬 Chat App

Bu proje, **Node.js**, **Express** ve **Socket.io** kullanılarak geliştirilmiş, gerçek zamanlı (real-time) iletişim sağlayan bir sunucu taraflı sohbet uygulamasıdır.

---

## 🚀 Özellikler

* **Gerçek Zamanlı Mesajlaşma:** Socket.io kütüphanesi ile düşük gecikmeli, çift yönlü veri iletimi.
* **Modern Express Altyapısı:** Güncel Express sürümü ile optimize edilmiş sunucu yönetimi.
* **Kolay Dağıtım:** Hızlı kurulum ve çalıştırılabilir script yapısı.

---

## 🛠️ Kullanılan Teknolojiler

| **Express** | Web sunucusu ve API yönetimi | <br />
| **Socket.io** | Gerçek zamanlı WebSocket iletişimi | <br />
| **Node.js** | JavaScript çalışma ortamı |
